# 评分卡 & 复盘模板

你发 `/复盘` 时，复盘师会用这套标准打分。也可以自己用它对照真实聊天记录复盘。

---

## 5 维评分卡（各 0–10，满分 50）

| 维度 | 看什么 | 高分长这样 | 低分长这样 |
|---|---|---|---|
| **1. 框架与心态** | 有没有需求感、谁在主导节奏 | 稳、不卑不亢、自己的节奏 | 讨好、秒回连发、患得患失 |
| **2. 话题与展开** | 能否打开并延伸话题 | 抓钩子深入、有画面有故事 | 查户口、不停换新问题、尬 |
| **3. 情绪价值** | 有没有调动情绪/张力 | 好玩、好奇、心动、推拉有度 | 平、闷、问答机器人 |
| **4. 节奏与邀约时机** | 推进是否踩点 | 窗口一到自然推进/邀约 | 错过窗口 或 太早强推 |
| **5. 自然度与真诚** | 像不像真人、有没有油腻感 | 真诚、有分寸、记得细节 | 套路感重、油、用力过猛 |

**评级参考**：40+ 优秀 / 30–39 良好 / 20–29 及格、有明显短板 / <20 重点回炉。

---

## 复盘输出格式（复盘师会这样给你）

```
【本段评分】框架 _/10 · 话题 _/10 · 情绪 _/10 · 节奏 _/10 · 自然 _/10 ｜ 总分 _/50（评级）
她的兴趣值变化：起 __ → 终 __（一句原因）

✅ 做得好（2 条）
- ……（对应原则：___）

⚠️ 要改（2 条）
- 问题：你说的「……」
  更好：「……」
  为什么：……（对应原则：___）

🎯 下一段刻意练习目标：……（一个就够）
```

## 自我复盘模板（练完/真实聊天后填，3 分钟）

1. 这次最好的一句话是？为什么有效？
2. 最想收回的一句话是？当时更好的说法？
3. 兴趣值是哪一刻往上 / 往下走的？
4. 我有没有需求感冒头的瞬间？
5. 下次只改 **一个** 点，是什么？

> 进步来自「每次只盯一个点刻意练」。别想一次全改对，刷量+复盘，自然成型。
