## 一、统一方法论：九套课的共同骨架

不管哪套课，剥到底层都是两句话：**价值是地基，推进关系是主线。** 其余所有技巧都挂在这两点上。

**1. 价值是吸引的大前提。** 女人长期会被有价值、有能力的人吸引（"慕强"）。价值分三类：

- **硬价值**——生存价值（钱、能力、社会地位、权力）与外显价值（颜值、身材、带出去有面子）。
- **软价值**——内在价值（思想、谈吐、见识、性格魅力）。
- **情绪价值**——你能给对方带来的情绪波动与体验，是聊天阶段最可操作、回报最快的一类。

关键认知：**吸引来自"对方的需求"，而不是单纯"你拥有什么"。** 同样的价值，传递给需求匹配的人才有效。所以先认清自己的价值、锁定相对匹配且竞争少的目标，再把价值通过**朋友圈 + 聊天**传递出去。

**2. 推进关系是聊天的总纲。** 和女生互动最重要的是持续往前推进（搭讪→线上→线下→升级），你不推就会被别人推走。推进本身就是价值感与主导性的体现；推得不好被拒绝也是好的反馈。怕被拒绝的唯一解法是大量实操。

**3. 聊天的本质 = 交互 + 调动情绪。** 聊天是有来有回的交互，不是单方面演讲；核心目的不是把话说漂亮，而是给对方制造情绪波动。死水一潭等于出局。五大雷区要避开：尬聊死（查户口）、自嗨死（自说自话/急着约）、表白死（过早表白）、炫耀死（不熟时晒房晒车）、吐槽死（倒负能量）。

**4. 全流程漏斗（本手册主线）：**

> 认知心态 → 展示面（朋友圈/形象） → 认识·搭讪 → 开场·破冰 → 聊天·吸引 → 推进·升级 → 邀约 → 约会 → （亲密推进） → 长期/挽回

几套课对这条线各有侧重的模型，互为补充：
- **M3 模型（三阶段九步）**：吸引 → 建立舒适感与联系感 → 诱惑。先判断自己处在哪个阶段，再用对应方法推进到下一阶段。
- **竞速逻辑**：先筛选（绿灯/黄灯/红灯），再用最高效率建立连接、尽快把人约出来见面。
- **五步流程**：搭讪 → 朋友圈/展示面 → 聊天框架 → 评估筛选 → 收尾升级。

**5. 高频术语速查**（全书反复出现）：
推拉、冷读、打压、误解/曲解、性张力、暧昧、废物测试（女生的试探）、惯例（即用话术/行动模板）、框架、需求感、巴纳姆效应（模糊描述让人对号入座）、洋葱四层模型（由表及里了解一个人）、复格（赋予认可）。
