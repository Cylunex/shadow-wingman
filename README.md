# shadow-wingman · 恋爱实战陪练系统

你扮演你自己，AI 同时扮演 **练习对象（女生）· 教练 · 复盘师 · 导演**，
陪你把搭讪、撩妹、聊天、约会练成肌肉记忆。无需安装，复制提示词到 Claude / GPT 即可开练。
