# shadow-wingman · 恋爱实战陪练系统

你扮演你自己，AI 同时分饰 **【她(练习对象)·教练·复盘师·导演】** 四个角色，陪你把搭讪、撩妹、聊天、邀约、约会练成肌肉记忆。她有一套**兴趣值**驱动的真实反应——会被逗笑、会冷淡、会抛废物测试、会因为你油腻/讨好/越界而失去兴趣；练完一键 `/复盘` 按 5 个维度打分、给更好的说法。

> 目标不是学套路，是把"有趣、自信、情绪稳定、真诚、懂分寸"练成自然反应，在现实里建立**真实、互相尊重的长期关系**。套路是脚手架，你这个人才是地基。

---

## 两种用法

### 用法 A：作为 skill（给 OpenClaw / Claude 等 AI agent）
把整个仓库（或 `SKILL.md` + `references/` + `progress/`）接入你的 agent。`SKILL.md` 顶部有 frontmatter（`name` / `description`），支持 skills 机制的 AI 会**自动识别**并在你说"陪我练撩妹""模拟一个高冷女生""练邀约""给我复盘打分"时触发；它按需加载 `references/` 里的深度料，不必一次全读。整套是**自包含、可移植**的，拷到别的技能目录即可用。

### 用法 B：直接粘进任意 AI 对话
把 [`SKILL.md`](SKILL.md) 整段（或 frontmatter 以下的正文）复制，粘贴进 Claude / ChatGPT 等任意对话框发送，即进入陪练模式。想加料就把 `references/` 里对应文件再贴进去。

---

## 三步上手

1. **开练**：触发 skill，或粘贴 `SKILL.md`。它会欢迎你，问你**想从哪个阶段练起 + 想要哪种练习对象**。
2. **像真聊天一样回**：你只说你真实会说的话（第一人称）。卡住、想看分、想换人时用下面的斜杠指令。
3. **复盘**：每段 `/复盘` 看 5 维打分与更好说法；**每次只改一个点**，刷量 + 复盘，自然成型。

### 常用指令
| 指令 | 作用 |
|---|---|
| `/开始 场景8` | 开一个指定场景（共 18 个，见 `references/scenarios.md`） |
| `/教练` | 叫场边教练给一条即时建议 |
| `/复盘` | 给这段按 5 维打分 + 改进 + 更好说法 |
| `/状态` | 看她当前兴趣值、绿/黄/红灯和原因 |
| `/换人 苏晴` | 换练习对象（7 种，见 `references/roles.md`） |
| `/难度 真实` | 调难度：简单 / 中等 / 真实 / 地狱 |
| `/提示` | 给几个回应方向（不直接给台词） |
| `/旁白` | 让导演推进剧情或丢个突发状况 |
| `/弹药` | 给一条贴情境、可直接照抄的话术 |

> 还能切语气：说"换狠一点 / 暖一点 / 正常点"。

## 目录结构

```
shadow-wingman/
├── README.md
├── SKILL.md            # 陪练引擎：可当 skill，也可整段粘进任意 AI 对话
├── references/         # 按需加载的深度料（全部自包含）
│   ├── knowledge-base.md   # 全流程实战 playbook（原理·打法·示范话术）
│   ├── ammo.md             # 话术弹药库（分类即用话术速查）
│   ├── signals.md          # 信号与雷区速查（绿黄红灯·IOI·身体语言·雷区）
│   ├── roles.md            # 7 张女生角色卡
│   ├── scenarios.md        # 18 个可直接开练的场景
│   └── scoring.md          # 5 维评分卡 · 阶段基准 · 兴趣值升降表
├── progress/          # 跨会话练习日志（私人数据，不入仓库；含 _template.md）
└── corpus/            # 背后的知识源库
    ├── 撩妹手册/           # 九套恋爱课融合成的 13 章实战书
    ├── 完整版教程/         # 九套课各自整理好的完整文字版
    ├── 转录文本/           # 课程原始转录稿
    └── routing-map.md     # 手册的内容路由表（编书/维护用）
```

> `corpus/` 是这套陪练背后的知识库；陪练系统的 `references/` 已把需要的精华**蒸馏复制进来、自成一体**，开练时不依赖 corpus。

## 练法建议

1. **每天 10–15 分钟**，刷 1–2 个场景，比偶尔猛练更有效。
2. 按阶段顺序（心态展示面 → 开场 → 文字聊天 → 邀约 → 约会升温）系统练；也可挑现实里最卡的那个反复刷。
3. 同一场景换不同角色（林夏 → 苏晴 → 周可可）再练，适应不同的人。
4. 现实里聊完，把真实对话贴回来让【复盘师】按同样标准复盘。
5. 想长期看进步，把每次复盘记进 `progress/`（复制 `_template.md` 起手），跟踪分数趋势与反复踩的雷。

## 底线（系统始终遵守）

真诚 > 套路；尊重对方意愿，"不"就是停；升温只讲信号与分寸、不写露骨；遇到明显没戏就建议止损、把精力放回自我提升。涉及未成年人、已婚出轨、骚扰跟踪、灌醉欺骗等一律拒绝。
